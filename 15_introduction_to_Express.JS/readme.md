# 🚀 Node.js + Express Basics

This project demonstrates the foundational structure of a Node.js application using Express.js, with a clear separation between the application logic and the server startup.

---

## 📚 What I Learned

- How to install and use Express.js
- How to structure a Node.js project using multiple files
- The difference between:
  - `app.js` (application setup)
  - `server.js` (server startup)
- How `module.exports` and `require()` work
- How to create a basic route in Express
- How to start a server and listen for requests

---

## 📁 Project Structure

│
├── app.js # Defines the Express application
├── server.js # Starts the server
